
## **📝 Interview Problem Set: DSA**

Passing Marks: 90/135

### **Module Marks**

* Each question max marks = 10
* Total = 135

---

### **EASY (3 questions, 10 marks each)**

**1️⃣ Question 1: Print Digits**
*"You are given a positive integer. I want you to print each digit of this number on a new line, starting from the most significant digit."*

**Example:**
Input: `5821`
Output:

```
5
8
2
1
```

**Marks:** 10 (5 for logic, 5 for implementation)

---

**2️⃣ Question 2: Count Digits**
*"Given an integer, return the count of digits in the number. Don’t convert it to a string. Also, think about negative numbers."*

**Example:**
Input: `-7645`
Output: `4`

**Marks:** 10 (4 for handling negatives, 6 for correct counting logic)

---

**3️⃣ Question 3: Reverse Number**
*"Write a function to reverse a given integer. If the reversed number exceeds integer limits (32-bit), return -1."*

**Example:**
Input: `1234` → Output: `4321`
Input: `-987` → Output: `-789`

**Marks:** 10 (5 for reversal logic, 5 for handling edge cases)

---

### **MEDIUM (3 questions, 15 marks each)**

**4️⃣ Question 4: Palindrome Check**
*"I want you to check if a number is a palindrome. A number is a palindrome if it reads the same backward as forward. Try solving it without converting to a string."*

**Example:**
Input: `121` → Output: `True`
Input: `123` → Output: `False`

**Marks:** 15 (7 for logic without string, 8 for correct implementation)

---

**5️⃣ Question 5: Armstrong Number**
*"Given a number, check if it is an Armstrong number. An Armstrong number is equal to the sum of its own digits raised to the power of the number of digits."*

**Example:**
Input: `153` → Output: `True`
Input: `123` → Output: `False`

**Marks:** 15 (8 for correct digit extraction, 7 for sum logic)

---

**6️⃣ Question 6: All Factors**
*"Print all factors of a given number in increasing order. Optimize it; don’t just loop from 1 to n. Think about the square root trick."*

**Example:**
Input: `28` → Output: `1 2 4 7 14 28`

**Marks:** 15 (10 for optimized logic, 5 for correct printing)

---

### **TRICKY / INTERVIEW LEVEL (3 questions, 20 marks each)**

**7️⃣ Question 7: Product of Digits Until Single Digit**
*"Write a function that multiplies the digits of a number repeatedly until the result is a single digit. Return that digit."*

**Example:**
Input: `234` → Process: 2×3×4=24 → 2×4=8 → Output: `8`

**Marks:** 20 (10 for logic, 10 for repeated multiplication handling)

---

**8️⃣ Question 8: Largest Palindromic Factor**
*"Given a number, find the largest factor of that number which is also a palindrome."*

**Example:**
Input: `120` → Factors: `1 2 3 4 5 6 8 10 12 15 20 24 30 40 60 120` → Largest palindromic factor: `6`

**Marks:** 20 (8 for factor logic, 7 for palindrome check, 5 for finding max)

---

**9️⃣ Question 9: Armstrong Numbers in Range**
*"Given a range `L` to `R`, print all Armstrong numbers in that range. Optimize for larger ranges up to 10⁶."*

**Example:**
Input: `100 200` → Output: `153`

**Marks:** 20 (10 for optimization, 5 for digit power sum, 5 for output formatting)
