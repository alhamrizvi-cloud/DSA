
# 🔢 Extract & Print Digits of a Number

## 🧠 Logic (Core Idea)

We process the number **digit by digit from right → left**:

* `n % 10` → gives **last digit**
* `n // 10` → **removes last digit**
* Repeat until `n = 0`

---

## 📌 Step Example (`n = 5873`)

| Step | n    | digit printed |
| ---- | ---- | ------------- |
| 1    | 5873 | 3             |
| 2    | 587  | 7             |
| 3    | 58   | 8             |
| 4    | 5    | 5             |
| 5    | 0    | stop          |

✅ Output:

```
3
7
8
5
```

---

## 🐍 Code (Cleaned)

```python id="k9f2zp"
# Extract and print digits of a number

n = 5873

while n > 0:
    digit = n % 10   # get last digit
    print(digit)
    n = n // 10      # remove last digit
```

---

## 🔁 Important Note

* Output is in **reverse order** (right → left)
* If you want **normal order (5 8 7 3)**:

  * store digits in a list and reverse it
  * OR reverse the number first

---

## 🧮 Mathematical Insight

* Each iteration reduces number by **1 digit**
* Total iterations = number of digits

[
\text{digits} = \lfloor \log_{10}(n) \rfloor + 1
]

---

## ⏱️ Time Complexity

* Loop runs once per digit

✅ **Time Complexity = O(log₁₀(n))**

---

## 💾 Space Complexity

* No extra storage used

✅ **Space Complexity = O(1)**

---

## ⚡ Quick Summary

* `% 10` → extract digit
* `// 10` → remove digit
* Output is reversed order
* **TC → O(log n)**
* **SC → O(1)**
