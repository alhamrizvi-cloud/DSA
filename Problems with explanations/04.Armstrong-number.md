# 🔢 Armstrong Number (e.g., 153)

## 🧠 Logic (Core Idea)

An **Armstrong number** is a number where:

[
\text{sum of (digits}^k\text{)} = \text{original number}
]

* `k` = number of digits

---

## 📌 Example (`n = 153`)

Digits = 1, 5, 3
Number of digits = 3

[
1^3 + 5^3 + 3^3 = 1 + 125 + 27 = 153
]

✅ So, **153 is an Armstrong number**

---

## 🧮 Mathematical Steps

1. Count digits → `k`
2. Extract each digit using `% 10`
3. Add `digit^k`
4. Compare with original number

---

## 🐍 Python Code

```python id="arm01"
# Check Armstrong Number

n = 153
num = n
k = len(str(n))   # number of digits
total = 0

while num > 0:
    digit = num % 10
    total += digit ** k
    num = num // 10

if total == n:
    print("Armstrong Number")
else:
    print("Not Armstrong")
```

## ⏱️ Time Complexity

* Loop runs once per digit
* Digits ≈ `log₁₀(n)`

✅ **Time Complexity = O(log n)**

---

## 💾 Space Complexity

* Only variables used (no extra DS)

✅ **Space Complexity = O(1)**

---

## ⚡ Quick Summary

* `% 10` → extract digit
* `// 10` → remove digit
* `digit ** k` → power
* Compare sum with original
* **TC → O(log n)**
* **SC → O(1)**
