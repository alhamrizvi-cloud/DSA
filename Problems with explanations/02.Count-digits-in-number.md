
# 🔢 Count Digits in a Number

## 🧠 Logic (Simple Understanding)

* We **keep removing the last digit** using `n // 10`
* Each time we remove a digit → we **increase count**
* Loop stops when number becomes `0`

---

## 📌 Step Example (`n = 5873`)

| Step | n    | count |
| ---- | ---- | ----- |
| 1    | 5873 | 1     |
| 2    | 587  | 2     |
| 3    | 58   | 3     |
| 4    | 5    | 4     |
| 5    | 0    | stop  |

✅ Total digits = **4**

---

## 🧮 Mathematical Logic

Number of digits in a number `n` is:

[
\text{digits} = \lfloor \log_{10}(n) \rfloor + 1
]

### Example:

[
\log_{10}(5873) \approx 3.76
]
[
\lfloor 3.76 \rfloor + 1 = 4
]

---

## 🐍 Your Code (Cleaned)

```python id="4u3k9y"
# Count how many digits in a number

n = 5873
count = 0

while n > 0:
    count += 1
    # This loop keeps removing digits until n becomes 0
    n = n // 10

print(count)
```

---

## ⏱️ Time Complexity

* Each loop removes **one digit**
* If number has `d` digits → loop runs `d` times

👉 Number of digits = **log₁₀(n)**

✅ **Time Complexity = O(log₁₀(n))**

---

## 💾 Space Complexity

* Only using:

  * `n`
  * `count`

✅ **Space Complexity = O(1)** (constant space)

---

## ⚡ Quick Summary

* `n // 10` → removes last digit
* Loop runs = number of digits
* Digits = `log₁₀(n)`
* **TC → O(log n)**
* **SC → O(1)**
