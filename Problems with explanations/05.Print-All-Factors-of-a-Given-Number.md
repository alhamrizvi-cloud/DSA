
We want **all factors of a given number** `num`.

* A factor `i` of `num` satisfies:

[
num \mod i = 0
]

* So, we check all integers `i` from 1 to `num`.
* Example: `num = 20`

[
\text{Check } i = 1,2,3,...,20
]

* Factors of 20 are numbers where `20 % i == 0`:

[
1, 2, 4, 5, 10, 20
]

* Visual logic (from your screenshot):

```
Check each number i:
i = 1 → 20 % 1 == 0 → yes
i = 2 → 20 % 2 == 0 → yes
i = 3 → 20 % 3 != 0 → no
i = 4 → 20 % 4 == 0 → yes
...
i = 20 → 20 % 20 == 0 → yes
```

---

## **2. Python Code (Brute Force)**

```python
def print_factors(num):
    result = []
    for i in range(1, num + 1):
        if num % i == 0:       # check if i is a factor
            result.append(i)   # add to the result list
    return result

# Example
num = 20
print(print_factors(num))  # Output: [1, 2, 4, 5, 10, 20]
```

---

## **3. Time Complexity (TC) and Space Complexity (SC)**

* **Time Complexity:** `O(N)`

  * We loop through numbers from `1` to `num` → total `num` iterations.
* **Space Complexity:** `O(K)`

  * Where `K` is the **number of factors** of `num` (for storing the result list).
